## You can customize mpv's options by setting it's config file (located in the mpv folder)

## HOWEVER, do NOT set the following options as Baka MPlayer sets them automatically
## and changing the following options could make Baka MPlayer unstable or crash.
## Setting other options should result in no problems.

# volstep
# msglevel
# osd-level
# no-keepaspect
# no-autosub
# cursor-autohide
# playing-msg
# status-msg
# volume
# wid
